Add contractors

If you want to add new company contractor, go to the Contractors and then choose Add contractors tab. You can add there the basic information about yours associates.

Contractors list

If you want to search information about companies we already prepared Contractors list.

Add invoice

To add an invoice, go to the Invoices and then choose Add invoice tab. If you want to add already existing contractor you can search him by NIP and all information about will be add automatically. If contractor does not exist, you can add him from this view. Moreover it provides PDF preview so you do not have go to separate tab. After completing the invoice data will available at Invoice details view.

Invoices list

At Invoices list tab you can search invoices and take a look at invoice details by clicking its number

Invoice detail

In this view you can see all information about chosen invoice (with PDF preview).
